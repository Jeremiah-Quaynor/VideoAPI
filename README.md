## Description
This is a VideoAPI created using flask and flask-restfull.


## Installation
```sh
    . .venv/bin/activate
    pip install -r requirements.txt
    export FLASK_APP=main.py
```

## Start the Server
```sh
    flask run 
```
or 
```sh
    python main.py
```


## Usage
```sh
    http://localhost:5000/
```
